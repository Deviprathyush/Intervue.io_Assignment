How to run

cd live-polling-system1/backend
npm install express socket.io
node index.js

cut src folder from frontend and paste it at other spot

cd .
cd live-polling-system1/frontend
npm create-react-app .
(replace the src folder)
npm install react-router-dom socket.io-client tailwindcss
npm start


after any changes to the code just do "npm start"
